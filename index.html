Workshops

6th February 2016

React on Rails

Vipul M Amler & Prathamesh Sonpatki

In this workshop, we will learn about ReactJS and using ReactJS with a Ruby on Rails app.

We will be building an app with React and Rails to learn about React as well as how to integrate it effectively with Rails.

7th February 2016

Deploying Rails apps using Chef

Sahil Muthoo & Mohit Thatte

A deepdive into Chef by building and deploying a Rails application.

Some of the topics covered in this workshop include:

  • Organize Chef code
  • Harness the Chef ecosystem / community
  • Set up Rails with Chef
  • Set up N-tier architectures
  • Deploy a Rails app to AWS
